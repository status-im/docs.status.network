# Testnet Faucets

To get started on Status Network testnet, you may need some testnet ETH or test assets.

:::warning Sepolia Testnet Sunset
The Sepolia-based testnet is subject to be sunsetted by the end of April 2026. Please migrate to the new **Status Network Hoodi Testnet** below. Refer to the [migration guide](https://status-im.notion.site/status-network-sepolia-testnet-deprecation-notice) for more information. 
:::

## Status Network Testnet ETH Faucet

The Status Network Hoodi Testnet faucet is available at [eth.faucet.status.network](https://eth.faucet.status.network).

### ETH Faucet Details

- **Faucet Contract**: [`0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81`](https://hoodiscan.status.network/address/0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81)
- **Amount**: 0.01 ETH per request
- **Cooldown**: One request per address per day
- **Condition**: Destination address must have at least 0.01 ETH on mainnet

### How to Get Testnet ETH

1. Visit [eth.faucet.status.network](https://eth.faucet.status.network)
2. Paste in your wallet address
3. Click "Request"
4. Wait for the ETH to appear in your wallet (usually takes a few seconds)

## Hoodi ETH (Required for Bridging)

To bridge assets to Status Network Hoodi Testnet, you'll first need Hoodi ETH. You can get it from:

1. [Hoodi Faucet](https://hoodi-eth.faucet.pk910.de/)

Once you have Hoodi ETH, you can bridge it to Status Network Hoodi Testnet using the [Status Network Bridge](https://bridge.status.network).

## Status Network STT Faucet

Status Test Token (STT) is the testnet equivalent of SNT. It is used as a test asset within the Status Network Testnet ecosystem.

### STT Faucet Details

- **Faucet Contract**: [`0x14c847ce7ab6c8002616176478b32C237fdF0cBb`](https://hoodiscan.status.network/address/0x14c847ce7ab6c8002616176478b32C237fdF0cBb)
- **Amount**: 10,000 STT per request
- **Cooldown**: One request per address per day

### How to Get STT

1. Visit [hub.status.network Stake page](https://hub.status.network/stake)
2. Connect your wallet and Sign-in
3. Click "Claim testnet SNT (STT)"
4. Confirm the transaction in your wallet
5. Wait for the STT to appear in your wallet (usually takes a few seconds)
