To get started on Status Network testnet, you may need some testnet ETH or test assets.

## Status Network Testnet ETH Faucet

The Status Network testnet faucet is available at [faucet.status.network](https://faucet.status.network).

### Details

- **Faucet Contract**: [`0x06338B70F1eAbc60d7A82C083e605C07F78bb878`](https://sepoliascan.status.network/address/0x06338B70F1eAbc60d7A82C083e605C07F78bb878)
- **Amount**: 0.01 ETH per request
- **Cooldown**: One request per address per day
- **Condition**: Destination address must have at least 0.01 ETH on mainnet

### How to Get Testnet ETH

1. Visit [faucet.status.network](https://faucet.status.network)
2. Paste in your wallet address
3. Click "Request"
4. Wait for the ETH to appear in your wallet (usually takes a few seconds)

## Sepolia ETH (Required for Bridging)

To bridge assets to Status Network testnet, you'll first need Sepolia ETH. You can get it from:

1. [Alchemy Sepolia Faucet](https://www.alchemy.com/faucets/ethereum-sepolia)
2. [Metamask Sepolia Faucet](https://docs.metamask.io/developer-tools/faucet/)

Once you have Sepolia ETH, you can bridge it to Status Network testnet using the [Status Network Bridge](https://bridge.status.network).

## Status Network STT Faucet

Status Test Token (STT) is the testnet equivalent of SNT. It is used as a test asset within the Status Network Testnet ecosystem.

### Details

- **Faucet Contract**: [`0x4Fb609F4a457f47B41D35Dd060447271F000120A`](https://sepoliascan.status.network/address/0x4Fb609F4a457f47B41D35Dd060447271F000120A)
- **Amount**: 10,000 STT per request
- **Cooldown**: One request per address per day

### How to Get STT

1. Visit [hub.status.network Stake page](https://hub.status.network/stake)
2. Connect your wallet and Sign-in
3. Click "Claim testnet SNT (STT)"
4. Confirm the transaction in your wallet
5. Wait for the STT to appear in your wallet (usually takes a few seconds)
