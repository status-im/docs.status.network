Status Network Hoodi テストネットを開始するには、テストネット ETH またはテストアセットが必要になる場合があります。

## Status Network Hoodi テストネット ETH フォーセット

Status Network Hoodi テストネットフォーセットは [faucet.status.network](https://eth.faucet.status.network) で利用できます。

### 詳細

- **フォーセットコントラクト**: [`0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81`](https://hoodiscan.status.network/address/0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81)
- **配布量**: リクエストあたり 0.01 ETH
- **クールダウン**: アドレスあたり 1 日 1 回のリクエスト
- **条件**: 送信先アドレスはメインネットに少なくとも 0.01 ETH を保有している必要があります

### テストネット ETH の入手方法

1. [faucet.status.network](https://eth.faucet.status.network) にアクセス
2. ウォレットアドレスを貼り付け
3. 「リクエスト」をクリック
4. ETH がウォレットに表示されるのを待つ（通常数秒かかります）

## Hoodi ETH（ブリッジに必要）

Status Network Hoodi テストネットにアセットをブリッジするには、まず Hoodi ETH が必要です。以下から入手できます：

1. [Alchemy Hoodi フォーセット](https://hoodi-eth.faucet.pk910.de/)
2. [Metamask Hoodi フォーセット](https://hoodi-eth.faucet.pk910.de/)

Hoodi ETH を入手したら、[Status Network ブリッジ](https://bridge.status.network) を使用して Status Network Hoodi テストネットにブリッジできます。

## Status Network STT フォーセット

Status Test Token (STT) は SNT のテストネット版です。Status Network Hoodi テストネットのエコシステム内でテストアセットとして使用されます。

### 詳細

- **フォーセットコントラクト**: [`0x14c847ce7ab6c8002616176478b32C237fdF0cBb`](https://hoodiscan.status.network/address/0x14c847ce7ab6c8002616176478b32C237fdF0cBb)
- **配布量**: リクエストあたり 10,000 STT
- **クールダウン**: アドレスあたり 1 日 1 回のリクエスト

### STT の入手方法

1. [hub.status.network Stake ページ](https://hub.status.network/stake) にアクセス
2. ウォレットを接続してサインイン
3. 「Claim testnet SNT (STT)」をクリック
4. ウォレットでトランザクションを確認
5. STT がウォレットに表示されるのを待つ（通常数秒かかります）

## サポート

フォーセットに問題が発生した場合：

- サポートについては [Telegram コミュニティ](https://t.me/statusl2) に参加してください。
