Status Network テストネットを開始するには、テストネット ETH またはテストアセットが必要になる場合があります。

## Status Network テストネット ETH フォーセット

Status Network テストネットフォーセットは [faucet.status.network](https://faucet.status.network) で利用できます。

### 詳細

- **フォーセットコントラクト**: [`0x06338B70F1eAbc60d7A82C083e605C07F78bb878`](https://sepoliascan.status.network/address/0x06338B70F1eAbc60d7A82C083e605C07F78bb878)
- **配布量**: リクエストあたり 0.01 ETH
- **クールダウン**: アドレスあたり 1 日 1 回のリクエスト
- **条件**: 送信先アドレスはメインネットに少なくとも 0.01 ETH を保有している必要があります

### テストネット ETH の入手方法

1. [faucet.status.network](https://faucet.status.network) にアクセス
2. ウォレットアドレスを貼り付け
3. 「Request」をクリック
4. ETH がウォレットに表示されるのを待つ（通常数秒かかります）

## Sepolia ETH（ブリッジに必要）

Status Network テストネットにアセットをブリッジするには、まず Sepolia ETH が必要です。以下から入手できます：

1. [Alchemy Sepolia フォーセット](https://www.alchemy.com/faucets/ethereum-sepolia)
2. [Metamask Sepolia フォーセット](https://docs.metamask.io/developer-tools/faucet/)

Sepolia ETH を入手したら、[Status Network ブリッジ](https://bridge.status.network) を使用して Status Network テストネットにブリッジできます。

## Status Network STT フォーセット

Status Test Token (STT) は SNT のテストネット版です。Status Network テストネットのエコシステム内でテストアセットとして使用されます。

### 詳細

- **フォーセットコントラクト**: [`0x4Fb609F4a457f47B41D35Dd060447271F000120A`](https://sepoliascan.status.network/address/0x4Fb609F4a457f47B41D35Dd060447271F000120A)
- **配布量**: リクエストあたり 10,000 STT
- **クールダウン**: アドレスあたり 1 日 1 回のリクエスト

### STT の入手方法

1. [hub.status.network Stake ページ](https://hub.status.network/stake) にアクセス
2. ウォレットを接続してサインイン (Sign-in)
3. 「Claim testnet SNT (STT)」をクリック
4. ウォレットでトランザクションを確認
5. STT がウォレットに表示されるのを待つ（通常数秒かかります）

## サポート

フォーセットに問題が発生した場合：

- サポートについては [Telegram コミュニティ](https://t.me/statusl2) に参加してください。
