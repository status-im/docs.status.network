# 사전예치 금고 컨트랙트

[Aragon](https://www.aragon.org/)과 협력하여 Status Network 사용자를 위한 사전예치 캠페인을 진행하고 있습니다.
이 페이지는 캠페인의 모든 중요한 컨트랙트 주소를 나열합니다.

:::note
금고의 토큰에 따라 Ethereum 메인넷 또는 Linea 메인넷에 배포됩니다
:::
