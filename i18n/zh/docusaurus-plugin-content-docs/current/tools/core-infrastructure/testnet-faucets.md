# 测试网水龙头

要开始在 Status Network Hoodi 测试网上进行开发，您可能需要一些测试网 ETH 或测试资产。

## Status Network Hoodi 测试网 ETH 水龙头

Status Network Hoodi 测试网水龙头可在 [faucet.status.network](https://eth.faucet.status.network) 访问。

### 详情

- **水龙头合约**: [`0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81`](https://hoodiscan.status.network/address/0x2D2ce8a43D63B7708451fB2A88D2A1060D242e81)
- **数量**: 每次请求 0.01 ETH
- **冷却时间**: 每个地址每天一次请求
- **条件**: 目标地址必须在主网持有至少 0.01 ETH

### 如何获取测试网 ETH

1. 访问 [faucet.status.network](https://eth.faucet.status.network)
2. 粘贴您的钱包地址
3. 点击“请求”
4. 等待 ETH 出现在您的钱包中（通常需要几秒钟）

## Hoodi ETH（跨链所需）

要将资产跨链到 Status Network Hoodi 测试网，您首先需要 Hoodi ETH。您可以从以下地方获取：

1. [Alchemy Hoodi 水龙头](https://hoodi-eth.faucet.pk910.de/)
2. [Metamask Hoodi 水龙头](https://hoodi-eth.faucet.pk910.de/)

获得 Hoodi ETH 后，您可以使用 [Status Network 跨链桥](https://bridge.status.network) 将其跨链到 Status Network Hoodi 测试网。

## Status Network STT 水龙头

Status Test Token (STT) 是 SNT 的测试网等效代币。它在 Status Network Hoodi 测试网生态系统中用作测试资产。

### 详情

- **水龙头合约**: [`0x14c847ce7ab6c8002616176478b32C237fdF0cBb`](https://hoodiscan.status.network/address/0x14c847ce7ab6c8002616176478b32C237fdF0cBb)
- **数量**: 每次请求 10,000 STT
- **冷却时间**: 每个地址每天一次请求

### 如何获取 STT

1. 访问 [hub.status.network Stake 页面](https://hub.status.network/stake)
2. 连接钱包并登录
3. 点击 "Claim testnet SNT (STT)"
4. 在钱包中确认交易
5. 等待 STT 出现在您的钱包中（通常需要几秒钟）
