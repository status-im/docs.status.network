Karma 在 Status Network 上提供无 Gas 交易特权和声誉系统。
本章节帮助您在应用中集成无 Gas 行为和基于层级的产品逻辑。

## 从这里开始

**处理交易？**

- 从 [无 Gas 交易集成](/build-for-karma/guides/gasless-integration) 开始，使用 `linea_estimateGas` 作为无 Gas 与付费 Gas 结果的唯一数据来源。

**按声誉等级控制功能？**

- 继续阅读 [声誉集成](/build-for-karma/guides/reputation-integration)，了解如何在链上读取 Karma 余额和层级、进行合约门控，以及设计层级感知的前端。
